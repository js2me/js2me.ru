# js2me.ru

Сайт на [Astro](https://astro.build)

## Разработка

```bash
npm install
npm run dev
```

Откройте [http://localhost:4321](http://localhost:4321).

## Сборка

```bash
npm run build
```

Статика в `dist/`. Превью: `npm run preview`.

## Блог

Посты — отдельные страницы в `src/pages/blog/`. Для динамического списка постов можно подключить [Content Collections](https://docs.astro.build/en/guides/content-collections/).
